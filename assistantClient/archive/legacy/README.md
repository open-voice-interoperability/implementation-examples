# Legacy Assistant Client Scripts

This folder contains historical or experimental scripts that are not part of the primary assistant client runtime.

## Archived Files

- `assistantClientLib.py` - older monolithic GUI variant
- `assistantClient-Pegasus.py` - prior alternate UI/protocol variant
- `known_agents-Pegasus.py` - legacy known-agents list used by Pegasus variant
- `cards-Pegasus.html` - legacy Pegasus HTML card template
- `testbrowser.py` - standalone HTML viewer helper script

## Active Runtime

Use `assistantClient.py` with:
- `event_handlers.py`
- `ui_components.py`
- `floor.py`
- `known_agents.py`
